
Materiais de laboratório

Os laboratórios de Química possuem diversos equipamentos, vidrarias, aparelhos e dispositivos que permitem a realização de inúmeras atividades com maior precisão e segurança.

Conheça os nomes dos principais materiais utilizados no laboratório e suas respectivas funções.

**1. Vidrarias de laboratório**

Esses materiais são feitos de vidro cristal ou temperado e podem variar de tamanho, capacidade suportada e função. Por isso, cada vidraria recebe uma aplicação específica.

| Vidraria                                                     | Imagem                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Balão de fundo chato**                                     | ![balão fundo chato](https://static.planejativo.com/uploads/novas/c90ee6cb87973fcb836ea1f0600fe556.jpg) |
| Utilizado no preparo de soluções, reações com desprendimento de gases ou aquecimento de líquidos. |                                                              |
| **Balão de fundo redondo**                                   | ![balão fundo redondo](https://static.planejativo.com/uploads/novas/133d2306dd524bc21745a4b8561fab8a.jpg) |
| Utilizado em processos de destilação, na separação de componentes de uma mistura líquida. |                                                              |
| **Balão de destilação**                                      | ![balão de destilação](https://static.planejativo.com/uploads/novas/298191ff4e3ec65115f31e255b43dedc.jpg) |
| Utilizado para aquecimento de mistura e separação dos compostos mais voláteis, que saem pelo tubo lateral. |                                                              |
| **Balão volumétrico**                                        | ![balão volumétrico](https://static.planejativo.com/uploads/novas/7c440c3b173886ec19facc852203f6a5.jpg) |
| Utilizado no preparo de soluções ou diluições com maior precisão. |                                                              |
| **Béquer ou Becker**                                         | ![béquer](https://static.planejativo.com/uploads/novas/74c1880ece380cdc937464355338b7a5.jpg) |
| Utilizado para medir volume, com pouca precisão, transferir líquidos e dissolver substâncias em experimentos ou misturas. |                                                              |
| **Erlenmeyer**                                               | ![erlenmeyer](https://static.planejativo.com/uploads/novas/c63efe54c0e11cc507d1c88c993766a8.jpg) |
| Utilizado para preparar soluções, armazená-las e realizar titulações. |                                                              |
| **Tubos de ensaio**                                          | ![tubos de ensaio](https://static.planejativo.com/uploads/novas/995465eeb5877d61fbb32c55462a720e.jpg) |
| Utilizados para reações onde os reagentes estão em pequenas quantidades. |                                                              |
| **Bureta**                                                   | ![bureta](https://static.planejativo.com/uploads/novas/a879255a04d5e60d936f4014b4cec65d.jpg) |
| Utilizada para realizar titulações e medir o volume de líquido que está sendo escoado. |                                                              |
| **Bastão de vidro**                                          | ![bastão de vidro](https://static.planejativo.com/uploads/novas/5bc7e1e18ff1e6c7126960a35919e5f7.jpg) |
| Utilizado para homogeneizar ou agitar soluções e auxiliar na transferência de líquidos de um recipiente para outro. |                                                              |
| **Condensador**                                              | ![condensador](https://static.planejativo.com/uploads/novas/29077cc56dd4806654e719f09b91761b.jpg) |
| Utilizado para resfriar gases separados no processo de destilação e torná-los líquidos. |                                                              |
| **Coluna de fracionamento**                                  | ![coluna de fracionamento](https://static.planejativo.com/uploads/novas/92ca14950d2dcfc11a8145755e366a7c.jpg) |
| Utilizada na destilação para separação dos componentes de uma mistura com diferentes pontos de ebulição. |                                                              |
| **Funil de bromo**                                           | ![funil de bromo](https://static.planejativo.com/uploads/novas/e6b81af517dad8953c8f052ec9bdf4fe.jpg) |
| Utilizado para separar líquidos imiscíveis através da decantação. |                                                              |
| **Funil de vidro**                                           | ![funil de vidro](https://static.planejativo.com/uploads/novas/3d9263f20adcec38446a0f2c0a16ff7e.jpg) |
| Utilizado, juntamente com papel de filtro, para reter sólidos que não estão dissolvidos em líquido. |                                                              |
| **Kitassato**                                                | ![kitassato](https://static.planejativo.com/uploads/novas/377f1258513185bfac709459a56ba98f.jpg) |
| Utilizado, juntamente com o funil de Büchner, para realizar filtrações a vácuo. |                                                              |
| **Placa de Petri**                                           | ![placa de petri](https://static.planejativo.com/uploads/novas/3879e129278271fc35e4dabe0c67fd30.jpg) |
| Utilizada como recipiente para cultivar micro-organismos, como as bactérias. |                                                              |
| **Pipeta graduada**                                          | ![pipeta graduada](https://static.planejativo.com/uploads/novas/e925b1629d6c2c07c98808d0afcbea34.jpg) |
| Utilizada para medir líquidos ou soluções com maior precisão e auxiliar na transferência para outros recipientes. |                                                              |
| **Pipeta volumétrica**                                       | ![pipeta volumétrica](https://static.planejativo.com/uploads/novas/3ad39fe748cd9ab005ebcffd836b13dd.jpg) |
| Utilizada para medir e transferir um volume fixo de líquido ou solução. Por isso, é mais precisa do que a pipeta graduada. |                                                              |
| **Proveta**                                                  | ![proveta](https://static.planejativo.com/uploads/novas/adcc1170f7812febb653a3709624a313.jpg) |
| Utilizada para medir e transferir volumes de líquidos e soluções, mas sem muita precisão. |                                                              |
| **Vidro de relógio**                                         | ![vidro de relógio](https://static.planejativo.com/uploads/novas/192ea6cbc17334b707e3fc9d9aaca4b5.jpg) |
| Utilizado para comportar pouca quantidade de amostra que será pesada, cobrir recipientes e evaporações em pequena escala. |                                                              |

**
\2. Equipamentos de laboratório**

Os equipamentos utilizados além de serem constituídos de diferentes materiais possuem aplicações específicas, podendo funcionar sozinhos ou em conjunto com outros materiais.

| Equipamento                                                  | Imagem                                                       |
| ------------------------------------------------------------ | ------------------------------------------------------------ |
| **Chapa aquecedora/Agitador**                                | ![chapa aquecedora](https://static.planejativo.com/uploads/novas/2092f2d8144ee6176baeeec829a5d7ad.jpg) |
| Utilizada para aquecer substâncias em um recipiente de maneira uniforme. Também tem a função de agitador para homogeneizar soluções enquanto aquece. |                                                              |
| **Barra magnética ou peixinho**                              | ![barra magnética](https://static.planejativo.com/uploads/novas/bafaf7c8fb0bf38c4b713375a181c36b.jpg) |
| Utilizada dentro de soluções que estão no agitador para serem homogeneizadas. |                                                              |
| **Almofariz e pistilo**                                      | ![almofariz e pistilo](https://static.planejativo.com/uploads/novas/79c3a5367d3881ecbc0c6e22808f0321.jpg) |
| Utilizados para trituração de pequenas amostras sólidas.     |                                                              |
| **Anel ou argola**                                           | ![anel ou argola](https://static.planejativo.com/uploads/novas/f5c511b7ae8d6ccee45d05de5afd2c31.jpg) |
| Utilizado, fixado ao suporte universal, para sustentar o funil de bromo durante a realização da decantação. |                                                              |
| **Balança de precisão**                                      | ![balança analítica](https://static.planejativo.com/uploads/novas/f7673d20f7111a5ffbca3fec7606629a.jpg) |
| Utilizada para medir a massa de materiais no laboratório com rigor. |                                                              |
| **Bico de Bunsen**                                           | ![bico de bunsen](https://static.planejativo.com/uploads/novas/59ac6b5c91da598c92724da1204773a4.jpg) |
| Utilizado para aquecer soluções, esterilizar objetos e realizar testes que necessitem de chama. |                                                              |
| **Deionizador de água**                                      | ![deionizador de água](https://static.planejativo.com/uploads/novas/01fdb74a0a014a6a871bb37693b4cb24.jpg) |
| Utilizado para remover íons na água, como cálcio (Ca2+) e magnésio (Mg2+), através da troca iônica. |                                                              |
| **Destilador de água**                                       | ![destilador de água](https://static.planejativo.com/uploads/novas/754bd97561e4e009872188d70cd0509b.jpg) |
| Utilizado para purificar a água, remover íons, impurezas e contaminantes que podem atrapalhar as análises químicas. |                                                              |
| **Cabine de fluxo laminar**                                  | ![cabine de fluxo laminar](https://static.planejativo.com/uploads/novas/14f184411a82245000c6294ed0c30b1d.jpg) |
| Utilizada para promover a recirculação do ar e promover um ambiente estéril. |                                                              |
| **Capela de exaustão**                                       | ![capela de exaustão](https://static.planejativo.com/uploads/novas/27646ce24565a6b63c65b59b8fbe8ba1.jpg) |
| Utilizada como barreira física para manipular materiais perigosos e eliminar os gases desprendidos. |                                                              |
| **Cadinho**                                                  | ![cadinho](https://static.planejativo.com/uploads/novas/d7a5ddc1573d96d0d24c98a047629d84.jpg) |
| Utilizado no aquecimento e fusão de sólidos, pois possui características refratárias e suporta elevadas temperaturas. |                                                              |
| **Cápsula de porcelana**                                     | ![cápsula de porcelana](https://static.planejativo.com/uploads/novas/b325166ed543020fcf867dc8c25a0e02.jpg) |
| Utilizada para concentrar e secar soluções através da evaporação de líquidos. |                                                              |
| **Cromatógrafo**                                             | ![cromatógrafo](https://static.planejativo.com/uploads/novas/24d47704d690711b1edac1ccd51bfcc8.jpg) |
| Utilizado para separar e identificar os componentes de uma mistura através da afinidade química. |                                                              |
| **Dessecador**                                               | ![dessecador](https://static.planejativo.com/uploads/novas/744a6356fd7206c1e9fbe5d3e8b9a25c.jpg) |
| Utilizado para remover a umidade dos materiais em atmosfera controlada e com a presença de agentes secantes. |                                                              |
| **Espectrofotômetro**                                        | ![espectrofotêmetro](https://static.planejativo.com/uploads/novas/639872ea3bbe879059b987cfab2e8414.jpg) |
| Utilizado para identificar e determinar a concentração dos componentes de uma amostra através da absorção de luz. |                                                              |
| **Estante para tubo de ensaio**                              | ![estante para tubos de ensaio](https://static.planejativo.com/uploads/novas/35694406dea9374e932ed48871b8bd86.jpg) |
| Utilizada para armazenar tubos de ensaio e como suporte para mantê-los em local fixo durante a utilização. |                                                              |
| **Espátula**                                                 | ![espátula](https://static.planejativo.com/uploads/novas/3cedea1bbe3b846cca9235ff26598da4.jpg) |
| Utilizada para manipular e transferir materiais sólidos de um recipiente para outro. |                                                              |
| **Estufa**                                                   | ![estufa](https://static.planejativo.com/uploads/novas/9a58533931a43b996fd529bc43735771.jpg) |
| Utilizada para secar e eliminar micro-organismos, por meio do calor, dos materiais em laboratório. |                                                              |
| **Funil de Büchner**                                         | ![Funil de Büchner](https://static.planejativo.com/uploads/novas/4fbcdc39c647c2cb494429126c61f124.jpg) |
| Utilizado, juntamente com o kitassato, para separar sólidos durante uma filtração a vácuo. |                                                              |
| **Pissete ou Pisseta**                                       | ![pisseta](https://static.planejativo.com/uploads/novas/cafa5457d581d1b1c461f5421ca6bb78.jpg) |
| Utilizada para armazenar líquidos, geralmente água destilada, e facilitar o manuseio na execução do trabalho. |                                                              |
| **Pinça metálica**                                           | ![pinça metálica](https://static.planejativo.com/uploads/novas/8ae32510a9858e05830d734402e81b69.jpg) |
| Utilizada para manusear objetos aquecidos, como o cadinho, para evitar queimaduras. |                                                              |
| **Pipeta Pasteur**                                           | ![pipeta pasteur](https://static.planejativo.com/uploads/novas/28fc334b5f9cfcebcf31e6e45eeda526.jpg) |
| Utilizada para transferir pequenas quantidades de líquidos através do gotejamento. |                                                              |
| **Pipetador tipo Pera**                                      | ![pipetador pera](https://static.planejativo.com/uploads/novas/56426e8a7c56473bcf3e76df36751ad6.jpg) |
| Utilizado para succionar líquidos para o interior das pipetas e liberá-los em um recipiente, de forma que o usuário não tenha contato com a substância. |                                                              |
| **Manta aquecedora**                                         | ![manta aquecedora](https://static.planejativo.com/uploads/novas/ce1117e8e9267358165acd7f499b4908.jpg) |
| Utilizada para aquecimento controlado de materiais.          |                                                              |
| **Mufla**                                                    | ![mufla](https://static.planejativo.com/uploads/novas/3efad423aeeadcb409a47a6a525eee7a.jpg) |
| Utilizada para calcinar amostras, ou seja, remover os compostos voláteis. |                                                              |
| **Papel de filtro**                                          | ![papel de filtro](https://static.planejativo.com/uploads/novas/5e8c1f872d936409503884dca9cf2a67.jpg) |
| Utilizado para reter materiais sólidos que não foram dissolvidos no líquido que passa por ele. |                                                              |
| **pHmetro**                                                  | ![pHmetro](https://static.planejativo.com/uploads/novas/35ca853b90461059eabeaa9465cd8515.jpg) |
| Utilizado para medir o pH de soluções através da condutividade. |                                                              |
| **Suporte universal**                                        | ![Suporte universal](https://static.planejativo.com/uploads/novas/343a9f858deb606923f284fb1f29b41e.jpg) |
| Utilizado para fixar as argolas que sustentam os equipamentos utilizados na vertical. |                                                              |
| **Tela de amianto**                                          | ![tela de amianto](https://static.planejativo.com/uploads/novas/29bb1afb693aba5aad71bca377c8e747.jpg) |
| Utilizada para sustentar o recipiente com a amostra durante o aquecimento. |                                                              |
| **Termômetro**                                               | **![termômetro](https://static.planejativo.com/uploads/novas/f391e39234f422c0414066f3d30d5290.jpg)** |
| Utilizado para medir ou acompanhar a temperatura de líquidos os soluções durante um experimento. |                                                              |
| **Tripé de ferro**                                           | ![tripé de ferro](https://static.planejativo.com/uploads/novas/802bbe00ff0692bb25a6370384651bde.jpg) |
| Utilizado para sustentar a tela de amianto durante o aquecimento de amostras. |                                                              |
